<b>-----------------------------ระบบฝั่งผู้ใช้ธรรมดา-----------------------------</b>
- อ่านกิจกรรม และ อัพเดท
- ซื้อสินค้า แลกของรางวัล
- เติมเงิน (มีแค่หน้าออกแบบไม่มีตัวระบบ)
- เติมโค๊ดรับแต้ม
- ติดต่อแอดมิน

- เพิ่ม Point และ เพิ่ม ลบ โค๊ดที่ใช้รับแต้ม
